# Marzban All-on-One-Port Auto Setup (HAProxy)

این اسکریپت به شما کمک می‌کند تا آموزش رسمی «راه‌اندازی همه‌چیز روی یک پورت» در مرزبان را به ساده‌ترین و سریع‌ترین شکل ممکن (تنها با چند کلیک) روی سرور خود پیاده‌سازی کنید. 

با استفاده از HAProxy، تمام ترافیک پنل، لینک‌های سابسکریپشن، کانفیگ‌های TLSدار و REALITY شما از روی **یک پورت استاندارد (443)** عبور خواهد کرد که باعث مخفی ماندن سرور از سیستم‌های فیلترینگ و کاهش مسدودی پورت‌ها می‌شود.

## 🚀 ویژگی‌ها
- نصب و کانفیگ خودکار `HAProxy`
- دریافت هوشمند نام دامنه‌ها (پنل و ریلیتی) و پورت‌های لوکال
- مسیریابی ترافیک بر اساس SNI (جلوگیری از اسکن شدن آی‌پی توسط کلادفلر)
- ویرایش خودکار فایل `.env` مرزبان جهت محدود کردن دسترسی مستقیم به پنل
- ریستارت خودکار سرویس‌ها

## ⚙️ نحوه نصب و اجرا

تنها کافیست با دسترسی `root` وارد سرور خود شوید و دستور تک‌خطی زیر را اجرا کنید:

```bash
bash <(curl -Ls [https://raw.githubusercontent.com/nikvpn-iran/auto-haproxy-marzban/main/setup.sh](https://raw.githubusercontent.com/nikvpn-iran/auto-haproxy-marzban/main/setup.sh))
